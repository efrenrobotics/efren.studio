I'm a believer in tools that disappear. The best tools are the ones you stop thinking about because they just work. Here's what I'm using right now.

## Editor

**VS Code** with a minimal setup. I use a dark theme, hide most UI chrome, and rely heavily on the command palette. The fewer distractions, the better.

Key extensions:
- Tailwind CSS IntelliSense
- ESLint
- Prettier
- GitHub Copilot

## Framework

**Next.js** is my default for any web project. The file-based routing, server components, and deployment story make it the most productive framework I've used.

Paired with **TypeScript** always. The type safety pays for itself within the first hour of any project.

## Styling

**Tailwind CSS** changed how I think about styling. Utility-first felt wrong at first, but once it clicked, I couldn't go back. The constraint of a design system built into your class names is genuinely freeing.

## Design

**Figma** for anything visual. I'm not a designer by title, but I think every frontend engineer should be comfortable in Figma. Understanding design intent makes you a better implementer.

## Terminal

**iTerm2** + **zsh** with a minimal prompt. I keep my dotfiles simple and version-controlled.

---

The common thread: **simplicity**. Every tool earns its place by reducing friction, not adding features.
